# Python Voice Assistant

This is a beginner-level voice assistant built using Python.

## Features
- Speech Recognition
- Text to Speech
- Tells Time and Date
- Opens YouTube and Google
- Wikipedia Search
- Plays YouTube Songs

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia API
- PyWhatKit

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python main.py
